# DESENVOLVIMENTO WEB III
* ***CURSO:*** TADS IFPR-Foz 
* ***PROFESSOR:*** Jefferson de Oliveira Chaves
* ***SEMESTRE:*** 3º
* ***HORA AULA:*** 80h
* ***HORA RELOGIO:*** 67h

* **Tecnologias: Java, Spring Framework**

### Objetivo:
Aprofundar os conceitos de desenvolvimento de sistemas web utilizando uma
linguagem de programação para a construção de páginas dinâmicas com
comunicação síncrona e assíncrona, mapeamento objeto-relacional para
persistência de dados e padrões de projetos voltados para desenvolvimento web.

### Ementa:
- Frameworks de desenvolvimento web(Java Spring)
- Padrões de projetos web e programação orientada a objetos
- chamadas assíncronas(AJAX)
- desenvolvimento de aplicações web n-camadas
- mapeamento objeto-relacional
- manipulação de eventos GET e POST
- utilização da sessão e cookies do navegador
- serialização de objetos.
